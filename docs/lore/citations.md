## Source Quality Guide

| Level | Description | Examples |
|-------|-------------|----------|
| **Official Canon** | Directly from rulebooks, cards, or official communications | Core rulebook, expansion rules |
| **Developer Confirmed** | Stated by Adam Poots in updates or interviews | Kickstarter updates, convention panels |
| **Strongly Implied** | Clear connections that can be drawn from official material | Cross-referenced story events |
| **Community Consensus** | Widely accepted theories with circumstantial evidence | Vibrant Lantern compilations |
| **Speculative** | Interesting theories without strong evidence | Fan theories, forum discussions |
