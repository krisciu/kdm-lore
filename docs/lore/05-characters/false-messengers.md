# False Messengers

Ripples of "what might have been" ordain the fates of those who follow. Expand your Kingdom Death: Monster games with these icons from imagined worlds.
False Messengers is packaged in a uniquely designed graphical box and includes:
4x Highly detailed hard plastic survivor miniatures
4x Disorder cards
4x Pattern cards
4x Pattern gear cards
1x Pillar card
1x Settlement event card
4x Large matte art prints
1x Install guide
